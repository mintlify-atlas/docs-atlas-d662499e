# Welcome to Substratum

Substratum is a Terra configuration pack that transforms Minecraft into a completely underground experience. Instead of traditional surface biomes, you'll explore an expansive subterranean realm filled with diverse cave systems, each with unique terrain, resources, and challenges.

## What is Substratum?

Substratum is built on the [Terra](https://terra.polydev.org) world generation platform, creating a fully underground world that spans from bedrock to bedrock. The entire overworld becomes a massive cave system with interconnected biomes, each offering distinct visual styles, blocks, and exploration opportunities.

## Key Features

### Diverse Cave Biomes

Substratum includes 14 unique cave biomes, each with distinctive characteristics:

- **Honey Grotto** - Honeycomb-filled caves with giant bee nests
- **Frozen Caverns** - Icy caves with blue ice, powder snow, and icicles
- **Molten Passages** - Lava-filled tunnels with magma blocks and basalt
- **Fungal Undergrowth** - Mushroom-covered caves with giant fungi
- **Amethyst Hollow** - Crystalline caves with amethyst clusters
- **Subterranean Mesa** - Underground badlands with terracotta layers
- **Marble Mines** - Calcite-rich caves with unique formations
- **Granite/Andesite/Diorite Mines** - Stone variant cave systems
- **And more...**

<Note>
Coral Coves biome is disabled by default due to border blending issues. You can enable it by uncommenting the biome in `add_substratum_cave_biomes.yml`.
</Note>

### Customizable Configuration

Substratum offers extensive customization through the `substratum_meta.yml` file:

- **Biome Distribution Scale** - Control the size of cave biomes (default: 200)
- **Cave Scale** - Adjust overall cave system size (default: 100)
- **Height Ranges** - Configure min/max Y levels for cave generation
- **Strata Layers** - Customize deepslate and bedrock layer positions

### Compatible with Existing Packs

Substratum can be used in two ways:

1. **World Installation** - Generate complete underground worlds
2. **Drop-In Merge** - Add Substratum caves to existing Terra packs like OverworldConfig

## Known Issues

<Warning>
Be aware of these potential issues when using Substratum:

- **Spawn Location**: Players may spawn at the top bedrock layer (Y=319) depending on world generation method
- **Spawn Suffocation**: Initial spawn might occur in solid blocks - configure spawn points manually if needed
- **Coral Coves**: This biome has messy borders and is disabled by default for stability
</Warning>
